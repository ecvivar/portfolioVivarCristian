Sobre mí

Hola, mi nombre es Cristian. Actualmente no trabajo en el área de sistemas, pero hace tiempo que me interesa el desarrollo de software. Comencé formándome de manera autodidacta con cursos y, recientemente, decidí dar un paso más iniciando la tecnicatura para profundizar mis conocimientos. En este camino ya participé en el desarrollo de proyectos, como una aplicación mobile para consultar el clima y un e-commerce con sistema de usuarios, carrito de compras y pagos integrados.

Proyectos

Vector Design Graphic

Vector Design Graphic es una tienda online desarrollada con Python, Flask y MySQL. Cuenta con sistema de usuarios, carrito de compras, integración de pagos con Mercado Pago y un panel de administración para gestionar productos. El proyecto utiliza Flask en el backend, MySQL como base de datos y HTML/CSS con templates Jinja2 en el frontend.

Sundae Weather

Sundae Weather es una aplicación mobile desarrollada con React Native que permite consultar el clima de distintas ciudades, visualizarlas en un mapa y gestionar un listado personalizado. La aplicación utiliza APIs externas para obtener datos climáticos y geográficos, junto con SQLite para la persistencia local de datos.
