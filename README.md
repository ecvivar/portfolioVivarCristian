PORTAFOLIO PERSONAL - CRISTIAN VIVAR

Este proyecto se trata de una página web donde se presenta un portfolio personal que incluye información, habilidades y proyectos desarrollados, aplicando HTML y CSS con un diseño visual inspirado en una estética retro.


Tecnologías utilizadas

- HTML5  
- CSS3  
- Google Fonts (Orbitron y Press Start 2P)  
- Flexbox para layout responsivo  
- Git y GitHub para versionado  


Secciones del sitio

- Sobre mí  
- Proyectos  
- Habilidades  
- Contacto (formulario)  
- Películas favoritas  


Características

- Diseño responsive  
- Estética retro con efectos visuales  
- Navegación interna 
- Tarjetas interactivas con efectos hover  
- Integración de iconos e imágenes  


Cómo ejecutar el proyecto

Clonar el repositorio:
git clone https://github.com/ecvivar/portfolioVivarCristian

Abrir el archivo index.html en el navegador para visualizar el sitio.


Deploy

Aplicación publicada en Vercel:  
https://portfolio-vivar-cristian.vercel.app
